72×72 Grid

50 mm Zellgröße (für Navigation irrelevant, nur Info)

Rechtsklick = Hindernis entfernen

Shift + Rechtsklick = Ziel setzen

Automatischer A*-Pfad: Start → Waypoints in Reihenfolge → Ziel

Start = Grün

Waypoints = Blau

Ziel = Rot

Hindernisse = Schwarz
